# DESCRIPTION: 
This project focuses on building a simple AI chatbot using the Python programming language and Natural Language Processing techniques. The chatbot is designed to interact with users in a conversational manner by understanding basic text input and providing appropriate responses. The main objective of this project is to demonstrate how NLP can be used to simulate human-like conversation in a simple and efficient way.
The chatbot is implemented using the NLTK library, which is one of the most popular NLP libraries in Python. It helps in processing and analyzing text data. In this project, tokenization is used to break down user input into individual words so that the chatbot can identify keywords and respond accordingly.

Tools and Technologies Used
1. Python
2. NLTK (Natural Language Toolkit)
3. Visual Studio Code (for development)
4. Command Line / Terminal
Working of the Chatbot
The chatbot operates using a rule-based approach. It continuously takes input from the user through the terminal and processes it using NLP techniques. The user input is first converted to lowercase and then tokenized into words. Based on the presence of specific keywords such as “hello”, “name”, “help”, or “weather”, the chatbot generates predefined responses.
The chatbot runs in an infinite loop, allowing continuous interaction until the user types “bye”, which exits the program. Conditional statements (if-elif-else) are used to match user input with known patterns and generate responses. If the chatbot does not understand the input, it responds with a default message indicating that it could not process the query.

Features
1. Interactive text-based chatbot
2. Uses NLP tokenization for input processing
3. Handles multiple types of queries (greetings, help, general questions)
4. Continuous conversation using loop
5. Exit command to end chat session
6. Beginner-friendly implementation
How to Run
1. Install nltk:
   pip install nltk
2. Run the program:
   python chatbot.py
3. Start chatting in terminal
